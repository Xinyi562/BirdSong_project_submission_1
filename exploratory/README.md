# Development & Exploratory Scripts
This folder documents tests and adjustments process for the bird song dataset. It includes:
* **Initial Drafts:** Early versions of audio filtering and metadata processing.
* **Experimental Scripts:** Component testing (e.g., sound playback, signal visualization).
* **Research Notes:** I put some comments in my native language, as it increases efficiency and accurateness when iterating.
*Note: These scripts are for historical reference and may not be fully runnable in the final environment. Please refer to the 'first' 'second' folder for the formal submission.*
