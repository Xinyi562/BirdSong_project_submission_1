# BirdSong Project
(Currently under construction...)

### Project Structure:
This project consists of three parts:

- Step One: Data Filtering (`first`)  
- Step Two: Visualize Frequency Distribution for Each Bird (`second`)  
- Step Three: Visualize Frequency Distribution in Geographic Context

### Important Note:
Please start by reading the README in the `first` folder, which covers Step One.  
Step Two builds upon the outputs of Step One (e.g., metadata files).  
If you attempt to run Step Two without completing Step One, the code will not work, because some necessary files are not included in this repository (e.g.`filtered_metadata_preview.csv`).
